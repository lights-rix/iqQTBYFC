# 项目简介

## 核心功能
- 基于YOLOv8的吸烟者检测系统，支持识别"person"和"smoke"两类目标
- 提供完整源码、数据集(2300张图片)、训练好的模型及训练结果
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供用户界面，可实时显示目标位置、目标总数、置信度等信息
- 支持自定义置信度和IOU阈值
- 支持选择显示标签和原图
- 支持选择检测模型
- 支持批量检测功能，可直接查看所有检测结果
- 支持检测结果保存

## 适用场景
- 公共场所吸烟行为监测
- 安全监控与行为识别
- 计算机视觉研究与教育
- 图像/视频目标检测应用开发

## 技术特点
- 采用Python 3.9开发，依赖OpenCV-Python、PyQt5、Ultralytics、Torch1.9等库
- 基于YOLOv8深度学习框架进行目标检测
- 提供完整的项目运行教程和环境配置说明
- 包含UI界面源文件，支持图形化操作
